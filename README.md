# 演算法程式作業 售貨員旅行解決器
本專題要解決的問題是售貨員最短路徑問題。<br>
使用的是Flutter與Dart，可以用於多種平台上，本次實作對象是iOS系統和Android系統裝置。<br>
可以輸入多個地址，程式會自動轉換成經緯度，並且計算每個地點的路徑<br>
最終提供用戶一個最短的拜訪路徑，除了提供文字上的指示以外，<br>
更以Google Cloud Platform Maps API顯示位置點，並且可視化最短路徑。<br>
